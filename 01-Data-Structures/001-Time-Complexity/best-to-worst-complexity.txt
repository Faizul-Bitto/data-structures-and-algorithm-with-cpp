For N = 1000

Constant O(1)           -> 1
Logarithmic O(log N)    -> 10
Square root O(sqrt N)   -> 32
Linear O(N)             -> 1000
Linearithmic O(N log N) -> 10000
Quadratic O(N*N)        -> 1000000

Best to Worst :

O(1)       ->       O(log N)     ->       O(sqrt N)       ->       O(N)       ->       O(N log N)       ->       O(N*N)
